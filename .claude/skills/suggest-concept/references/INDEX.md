# Visual Reference Library Index

A curated collection of 45 best-in-class products across 15 verticals, each with 3 screenshot references. Use this index to find the most relevant visual references when generating design concepts.

---

## How to Use This Index

1. **Match the user's concept to one or more verticals** based on the domain, target users, and product type
2. **Select 2-3 products** from the matched verticals that best align with the concept's UI needs
3. **Read the screenshots** to understand the design patterns, layout approaches, and visual language
4. **Cite the references** when presenting concepts to the user — explain what was borrowed and why
